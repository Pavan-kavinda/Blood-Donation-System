# Blood Donation Management System - Backend

## 🩸 Project Overview
This is a Node.js and Express-based backend application designed to manage blood donor registrations. It allows users to register donors and retrieve donor information via RESTful APIs.

## 🛠️ Technologies Used
- **Node.js**: Server-side runtime.
- **Express.js**: Backend framework for building APIs.
- **MongoDB & Mongoose**: Local database (MongoDB Compass) for data storage.
- **Postman**: Used for API testing and validation.

## 📂 Project Structure
- `models/`: Database schemas (Donor model).
- `controllers/`: Logic for handling requests.
- `routes/`: API endpoints definitions.
- `server.js`: Main entry point.
